# logs-ai-reporting

AI logs reporting solution.

## Persistent volumes
- `/app/data` — input data  
- `/app/outputs` — generated reports/logs  
- `/var/lib/postgresql/data` — Postgres cluster  

## Required environment variable
- `GEMINI_STUDIO_API_KEY` — Gemini Studio key for LLM access
